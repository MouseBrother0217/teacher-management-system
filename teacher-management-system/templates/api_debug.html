师资管理系统 RESTful API v1 - 在线调试

GET /api/v1/teachers 获取教师列表（支持分页、筛选、排序）
参数类型必填说明
pageint否页码，默认 1
per_pageint否每页条数，默认 20，最大 100
keywordstring否关键词搜索（姓名/领域/职称/单位）
in_stockstring否是否入库：是/否
approval_statusstring否审批状态：已通过/待审核
sortstring否排序：score/count/name
